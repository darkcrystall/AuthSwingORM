# Sistema de Login em Java – Tecnologias e Importância

Este projeto foi desenvolvido em **Java Desktop**, utilizando **Swing** para a interface gráfica e tecnologias modernas de persistência e segurança.  
O foco principal é demonstrar boas práticas de desenvolvimento, organização em camadas e proteção de dados sensíveis.

---

## Tecnologias Utilizadas e Sua Importância

### Java

A linguagem principal do projeto.  
O Java é amplamente utilizado no mercado por sua:

- Portabilidade (Write Once, Run Anywhere)  
- Robustez e segurança  
- Forte orientação a objetos  
- Ampla adoção corporativa  

É uma excelente escolha para aplicações desktop e sistemas corporativos.

---

### Swing

Biblioteca nativa do Java para construção de interfaces gráficas (GUI).

**Importância:**

- Permite criar aplicações desktop completas  
- Não depende de tecnologias web  
- Ideal para sistemas internos e acadêmicos  
- Ajuda a compreender eventos, componentes visuais e arquitetura MVC  

---

### ORM (Object-Relational Mapping)

O ORM é um conceito que permite mapear objetos Java para tabelas do banco de dados.

**Importância:**

- Reduz a necessidade de SQL manual  
- Facilita manutenção  
- Melhora organização do código  
- Aumenta produtividade  

---

### Hibernate

Implementação de ORM utilizada no projeto: Hibernate.

**Importância:**

- Automatiza o mapeamento objeto-relacional  
- Gerencia sessões e transações  
- Reduz código repetitivo  
- Amplamente utilizado no mercado corporativo  

---

### JPA (Jakarta Persistence API)

Especificação utilizada para padronizar a persistência: Jakarta Persistence API.

**Importância:**

- Define padrões para mapeamento de entidades  
- Permite trocar a implementação (Hibernate, EclipseLink, etc.)  
- Facilita manutenção e escalabilidade  
- Segue boas práticas de arquitetura  

---

### BCrypt

Biblioteca utilizada para criptografia de senhas: BCrypt.

**Importância:**

- Gera hash seguro da senha  
- Inclui salt automático  
- Protege contra ataques de força bruta  
- Evita armazenamento de senha em texto puro  

É uma prática essencial em qualquer sistema de autenticação.

---

### Regex (Expressões Regulares)

Utilizado para validação de dados como email e senha.

**Importância:**

- Garante padrão correto de entrada  
- Evita dados inválidos no banco  
- Aumenta a segurança  
- Melhora a experiência do usuário  

---

## Importância do Projeto

Este projeto demonstra:

- Aplicação prática de autenticação segura  
- Uso de criptografia real em sistemas  
- Integração entre interface gráfica e banco de dados  
- Organização em camadas (Model, DAO, Service, View)  
- Boas práticas de desenvolvimento Java  

Além disso, é um excelente projeto para:

- Portfólio profissional  
- Demonstração de conhecimento em Java Desktop  
- Base para evolução para sistemas maiores  

---

## Conclusão

O sistema combina interface gráfica, persistência robusta e segurança de dados, aplicando tecnologias amplamente utilizadas no mercado.

Ele demonstra domínio de:

- Desenvolvimento Desktop com Java  
- Persistência com Hibernate e JPA  
- Segurança com BCrypt  
- Validação de dados  
- Arquitetura organizada  

Um projeto simples em conceito, mas completo em fundamentos técnicos.
